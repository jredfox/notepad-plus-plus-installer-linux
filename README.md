# notepad-plus-plus-installer-linux
 Install Notepad++ On Linux the Easy Way!
 
# Usage:
- Download Notepad++ from their official website https://notepad-plus-plus.org/
- sudo apt update && sudo apt install zip unzip icoutils wine
- bash `InstallNotepad++.sh`

# System Requirements
- Intel 64 bit processor (x86_64, x64, AMD64) or Intel 32 bit processor (x86, x86_32, i?86, i??x86)
